# ラテカピュータ 復活プロジェクト 目論見書

## コンセプト

1979年にシャープから発売された「ラテカピュータ」（ラテカセ＋コンピュータ）の現代版復活プロジェクト。
製造台数200台の幻のマルチメディアデバイスを、現代の技術で再解釈・拡張する。

素体となるラテカセ完動美品に対し、オリジナルと同様に下部に拡張筐体を追加。
内部に現代のコンピューティング・メディア・通信デバイスを詰め込み、
ESP32デバイスがそれらを統合制御するUIハブとして機能する。

---

## システム構成

### ラテカセ筐体内部：XIAO ESP32-C6

ラテカセの物理操作をデジタル化し、拡張筐体側へ伝達する。

**入力**
- チューニングドラム（回転角・位置読み取り）
- 各種物理スイッチ状態

**出力**
- Waveshare ESP32-C6 Touch LCDへイベント送信（ESP-NOW）

### 拡張筐体：Waveshare ESP32-C6 Touch LCD

メインUIハブ。タッチ画面でシステム全体を操作・制御する。

**制御対象**
- 各デバイスへの電源制御
- 赤外線ユニット経由でのHDMI切替器・TVチューナ制御
- Raspberry Pi 4との通信によるSDR周波数制御
- 地デジチャンネル切り替え・HDMI入力切り替え

**Matter対応**
- ESP32-C6のThread/Matter機能を活用
- このデバイス自体をMatter対応ホーム家電として現代のスマートホームネットワークに接続

### 拡張筐体内 その他ハードウェア

| デバイス | 役割 |
|---|---|
| スティック型Windows PC | メインコンピュータ |
| Raspberry Pi 4 | SDR制御 |
| SDR（ソフトウェアラジオ） | ラジオ受信 |
| TVチューナ | 地デジ受信 |
| HDMI切替器 | 映像入力切り替え |
| フロッピーディスクドライブ | レトロメディア対応 |
| MOディスクドライブ | レトロメディア対応 |
| ブルーレイドライブ | 光学メディア対応 |
| メモリカードリーダ | 各種メモリカード対応 |
| パラレルインタフェース | レトロ周辺機器対応 |

## デバイス間通信

```
XIAO ESP32-C6（ラテカセ内）
    │ ESP-NOW
    ▼
Waveshare ESP32-C6 Touch LCD（拡張筐体）
    │ UART/USB/WiFi
    ▼
Raspberry Pi 4
    │ USB
    ▼
SDR
```

## 開発フェーズ

### Phase 1: 基盤構築
- [ ] XIAO C6：チューニングドラム・スイッチ読み取り

### Phase 2: デバイス制御
- [ ] 電源制御回路との接続
- [ ] 赤外線送信によるHDMI切替・チューナ制御
- [ ] Raspberry Pi 4との通信・SDR制御

### Phase 3: 統合
- [ ] チューニングドラム操作 → SDR/TV連動
- [ ] UIからの一元操作

### Phase 4: Matter対応
- [ ] Matter/Thread設定
- [ ] スマートホーム連携
